# Корпоративный портал IT-аутсорсинговой компании

## О проекте

Корпоративный портал для компании, занимающейся IT-аутсорсингом и системным администрированием. Объединяет учёт заявок (service desk), инвентаризацию оборудования (CMDB), базу знаний, AI-ассистента и аналитику по заявкам/времени.

- **Техническое задание:** [docs/TZ.md](docs/TZ.md) — живой документ, дополняется по ходу проекта
- **Журнал решений и изменений:** [CHANGELOG.md](CHANGELOG.md)

## Контекст компании

- Профиль: IT-аутсорсинг, системное администрирование
- Офлайн-клиенты: обслуживание физической техники — ПК, ноутбуки, серверы, сетевое оборудование, телефония, периферия и др.
- Облачные клиенты: удалённые рабочие столы, 1С в облаке, виртуальные ресурсы и др.

Подробности и обоснование — в [docs/TZ.md](docs/TZ.md).

## Сервер для деплоя

- Хост: 10.3.0.88 (`vrtvs01`), SSH-доступ есть (учётные данные не хранятся в репозитории)
- ОС: Ubuntu 24.04.4 LTS (x86_64)
- Ресурсы: 2 vCPU, 3.8 GB RAM, диск 19 GB (свободно ~11 GB)
- Пока ничего не установлено (Docker, Node.js, PostgreSQL, nginx отсутствуют), есть git и python3
- sudo для vrtadmin требует пароль (понадобится при установке зависимостей)
- Подробности — в [docs/TZ.md](docs/TZ.md), раздел 5.1
